New to Angular?

These will help you to get started

Node & NPM

Make sure you have node v12.x and npm v6.x or greater installed by running node -v and npm -v

Don't have it? Everything you need to know about downloading is here.

Update .npmrc for our private nexus npm repository

Set the registry to nexus

npm config set registry=https://nexus.nimhda.org/repository/nda-npm/

Add your nexus user to npm

npm adduser --registry=https://nexus.nimhda.org/repository/nda-npm/

Enter your nexus username, password and email

Angular CLI

Install globally

npm install -g @angular/cli

NDA Schematics

Install globally

npm install -g @nda/schematics

Create new NDA angular app

ng new [name] -c @nda/schematics

where [name] is the name of your new app

Set cms content id

Open src/environments/environment.ts and src/environments/environment.prod.ts

Add cms configuration and content ids. Leave as empty string if project configuration is not used.

Start in dev mode

npm run start

Build for production

npm run build

Update nda libraries to latest release

npm update @nda/{angular,ui}
